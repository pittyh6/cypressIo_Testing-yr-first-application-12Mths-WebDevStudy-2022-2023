## What I learned/used 
### Cypress 
* Cypress
    * Install Cypress
        * npm install cypress --save-dev
    * open Cypress
        * npx cypress open
    * Config
        * select e2e test
        * click continue
        * select Chrome
        * click Start E2E testing in Chrome
        * create new empty spec
        * rename the file to home
            * cypress/e2e/home.cy.ts
            * click Create Spec
        * click Okay, run the spec
